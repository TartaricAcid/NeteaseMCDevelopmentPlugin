# MC Studio Helper（MC Studio 助手）

<img src="https://img.shields.io/badge/license-MIT-green" alt="License">

本插件为网易我的世界中国版（Minecraft 中国版，网易代理）提供组件开发与调试支持，面向 PyCharm 软件使用的插件。

> 注意：本插件与网易官方无任何从属关系，仅供个人学习与开发使用。

## 简短说明

- 本插件旨在简化在本地环境下为网易我的世界中国版编写、运行与调试扩展/组件的工作流。
- 当前为社区/个人维护的非官方工具，功能仍在持续开发中。

## 主要功能（含画饼部分）

- 已完成：添加运行配置，支持直接使用 MC Studio 运行
- 进行中：新建预设组件模板、代码补全与提示
- 计划：支持调试断点（计划）
